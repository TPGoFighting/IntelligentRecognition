# IntelligentRecognition 准确性测试指南

## 测试目标
验证通用形状识别系统在隧道点云数据上的检测准确性，使用图形化界面进行交互式测试。

## 测试环境要求

### 硬件要求
- GPU: NVIDIA GPU (推荐) 或 CPU
- 内存: 8GB+ RAM
- 存储: 1GB+ 可用空间

### 软件要求
- Python 3.9+
- 依赖包: 见 `requirements.txt`
- 操作系统: Windows 10/11, Linux, macOS

## 环境设置步骤

### 1. 创建虚拟环境
```bash
conda create -n intelligent_recognition python=3.9
conda activate intelligent_recognition
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 验证安装
```bash
python -c "import torch, open3d, pyvista, PyQt5; print('所有依赖安装成功')"
```

## 测试数据准备

### 可用测试数据
项目包含4个隧道点云测试文件：
1. `test_data/tiny_tunnel.ply` - 小型隧道 (51KB)
2. `test_data/small_tunnel.ply` - 小型隧道 (510KB)
3. `test_data/medium_tunnel.ply` - 中型隧道 (2.5MB)
4. `test_data/large_tunnel.ply` - 大型隧道 (10.2MB)

### 测试场景选择
建议从 `tiny_tunnel.ply` 开始测试，逐步升级到更大文件。

## 准确性评估指标

### 定性评估
1. **可视化检查**:
   - 检测到的几何体是否与点云中的实际管道对齐
   - 圆柱体半径是否合理（典型隧道管道半径: 0.15-0.8米）
   - 是否存在误检（将背景识别为形状）
   - 是否存在漏检（实际管道未被识别）

2. **物理合理性**:
   - 圆柱体方向是否合理（隧道管道通常沿Z轴）
   - 尺寸是否符合实际工程标准
   - 位置是否在合理空间范围内

### 定量评估
1. **检测数量**: 与实际管道数量比较
2. **置信度**: 平均置信度应 > 0.5
3. **点云覆盖率**: 检测到的点数占总点数的比例

## 常见问题排查

### 问题1: 依赖安装失败
**症状**: 启动时显示 `ModuleNotFoundError`
**解决方案**:
```bash
# 单独安装有问题的包
pip install open3d
pip install pyvistaqt
pip install pyransac3d
```

### 问题2: 模型加载失败
**症状**: 日志显示模型文件不存在或格式错误
**解决方案**:
1. 检查模型文件路径: `models/universal/best_model.pth`
2. 确认文件存在且可读
3. 尝试加载其他模型: `models/cylinder_model/best_model.pth`

### 问题3: 点云加载失败
**症状**: 无法读取点云文件或显示为空
**解决方案**:
1. 确认文件格式受支持 (.ply, .pcd, .npz, .npy, .txt)
2. 检查文件完整性
3. 尝试其他测试文件

### 问题4: 检测结果不准确
**症状**: 误检、漏检或几何参数不合理
**解决方案**:
1. 调整体素大小参数（默认0.05）
2. 修改最大窗口数（默认200）
3. 尝试不同的配置：`config/cylinder_only_config.yaml`
4. 使用专用隧道检测模式
